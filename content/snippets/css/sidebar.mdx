## Code

```html
<div class="container">
  <div class="sidebar">
    Min: 150px
    <br />
    Max: 25%
  </div>
  <main>...</main>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: minmax(150px, 25%) 1fr;
  padding: 0;
  margin: 0;
}

.sidebar {
  height: 100vh;
  background: lightpink;
  font-size: 2rem;
  text-align: center;
}
```
