---
title: Dynamic Tag Name Props
description: Allow users to pass in a tag name prop to a component as a string.
category: react-components
humanReadableDate: May 28th, 2022
---

## Code

```jsx
function Component(props) {
  const { as = 'div', children } = props;

  const Tag = as;

  return <Tag>{children}</Tag>;
}
```

## Usage

```jsx
export default function App() {
  return (
    <Component as="section">
      {
        //    ...rest components
      }
    </Component>
  );
}
```
